FormulatePro Support

PDF annotation made simple for macOS

Getting Started

FormulatePro lets you open any PDF and add annotations on top of it. To get started:

  1. Open a PDF with File → Open
  2. Select a tool from the Tool Palette
  3. Click or drag on the document to add annotations
  4. Save your work as a .formulate file to preserve editability
  5. Export as PDF via File → Print → PDF → Save As PDF

Frequently Asked Questions

What tools are available?

FormulatePro includes: Arrow (selection/move), Ellipse, Rectangle, Freehand drawing, Text Area, Checkmark, and Stamp tools.

How do I change colors and line width?

Select an annotation, then use the Inspector panel to adjust stroke color, fill color, and line width.

How do I export my annotated document as a PDF?

Go to File → Print, click the PDF button in the lower-left corner, and choose Save As PDF.

Can I hide annotations when printing?

Yes. Select the annotation, then check Hide When Printing in the Inspector panel.

What keyboard shortcuts are available?

T = Text, M = Arrow, E = Ellipse, U = Rectangle, P = Freehand, X = Checkmark. Hold Cmd+Option to temporarily switch to the Arrow tool.

Does FormulatePro collect any data?

No. FormulatePro works entirely offline and does not collect, transmit, or share any data. See our Privacy Policy.
